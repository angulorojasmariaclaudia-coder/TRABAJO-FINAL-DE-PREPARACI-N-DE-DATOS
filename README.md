# TRABAJO FINAL DE PREPARACIÓN DE DATOS


Proyecto:

Preparación y selección de datos para el análisis de la severidad de los desastres naturales.

Autoras:

María Claudia Angulo

Curso:

Preparación de Datos

Maestro:

Máster en Ciencia de Datos y Aprendizaje Automático – Universidad de La Rioja

Descripción:

Este proyecto aplica  técnicas de preparación y selección de datos a un conjunto de datos histórico de desastres naturales, con el objetivo de analizar su impacto humano. A través de un proceso sistemático de exploración, limpieza y transformación de los datos, se prepara un dataset adecuado para la construcción de un modelo de clasificación que permita diferenciar distintos niveles de impacto asociados a los desastres.
